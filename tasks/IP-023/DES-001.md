# DES-001: Thiết kế Website & Visual Assets cho Dịch vụ Tái cấu trúc

## Mục tiêu

Thiết kế UI/UX cho trang dịch vụ đã tái cấu trúc trên cyberk.io (thay thế trang cũ) và toàn bộ visual assets phục vụ marketing + sales.

## Bối cảnh

Cyberk đang tái cấu trúc dịch vụ từ catalog cũ (MVP Development, Tech Labs, Kick-starter Marketing, Community Development) sang catalog mới:

- **Product Design** — 3 gói ($5K / $7K / $8K)
- **Product Build** — 2 gói ($29K / $35K)
- **Tech Labs** — bảng giá theo role

Cần thiết kế trang web mới + tài liệu bán hàng thay thế hoàn toàn trang cũ.

## Việc cần làm

### 1. Website Pages (Figma)

- [ ] Trang **Services Overview** — landing page tổng hợp 3 dịch vụ, customer journey (Design → Build → Scale)
- [ ] Trang **Product Design** — highlight 3 tiers, bảng so sánh, CEO advisory badge
- [ ] Trang **Product Build** — highlight 2 tiers, tech stack, timeline 30 ngày
- [ ] Trang **Tech Labs** — bảng giá roles, lợi thế Cyberk Framework
- [ ] Responsive design (Desktop + Mobile + Tablet)

### 2. Visual Assets

- [ ] Pricing cards bảng so sánh 3 gói Product Design (Basic / Pro / AIO)
- [ ] Pricing cards bảng so sánh 2 gói Product Build (Basic / Pro)
- [ ] Infographic: Customer Journey (Design → Build → Scale)
- [ ] Banner images cho social media (LinkedIn cover, Twitter/X header)
- [ ] Email header/banner cho thông báo dịch vụ mới

### 3. Sales Materials

- [ ] Sales deck PDF (10–15 slides) cho BD/broker gửi khách hàng
- [ ] One-pager cho mỗi service (1 trang A4, in được, gửi email được)

### 4. Brand Consistency

- [ ] Đảm bảo nhất quán với design system cyberk.io hiện tại
- [ ] Color coding cho các tiers: Basic / Professional / All-in-One / Pro
- [ ] Icon set cho deliverables (Figma, Landing Page, SEO, Monitoring...)

## Phối hợp

- Phối hợp **DEV** để code HTML sau khi Figma approved
- Phối hợp **MKT** để cung cấp visual assets cho social media
- Phối hợp **BD** để hoàn thiện sales deck

## Tài liệu tham khảo

- Service descriptions: `gitbook/services/product-design.md`, `product-build.md`, `tech-labs.md`
- Design standard: `gitbook/design-standard/design-standard.md`
- Website hiện tại: cyberk.io
