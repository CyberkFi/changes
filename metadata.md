# Internal Workspace — Metadata

Quy ước chung xem [[metadata-rule]].

---

## `type`

| Giá trị | Mô tả |
| ------- | ----- |
| `ip` | Kế hoạch cải tiến (Improvement Plan) |
| `strategy` | Chiến lược dài hạn (Strategy) |
| `readme` | File tổng quan, mục lục, hướng dẫn của thư mục |
| `kanban` | Bảng theo dõi tiến độ Kanban |
| `diary` | Nhật ký thực hiện IP/Strategy |
| `thinking` | Suy nghĩ, thảo luận, ghi chú cá nhân của CEO |
| `workflow` | Quy trình, hướng dẫn thực hiện |
| `template` | Mẫu tài liệu chuẩn |
| `goal` | Mục tiêu công ty, tầm nhìn |
