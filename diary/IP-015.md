#### SEO & Performance
*   Mỗi trang có: title, meta description, canonical URL, Open Graph, Twitter Card
*   Structured Data (JSON-LD): Organization + WebSite schema trên mỗi trang
*   `robots.txt` + `sitemap.xml` với 13 URLs
*   JS budget: 1.2KB (dưới giới hạn 2KB)
*   CSS total: 27KB (dưới giới hạn 50KB)

#### Quyết định thiết kế
*   **CSS-first interactions:** Accordion dùng `<details>/<summary>`, hamburger menu dùng checkbox hack, carousel dùng scroll-snap
*   **Fallback vanilla JS:** Chỉ cho header hide/show on scroll, How We Work auto-rotate, close mobile nav on link click
*   **Responsive:** Mobile-first, breakpoints 768px/1024px/1280px, fluid typography với `clamp()`
*   **Hero animation:** Thay WebGL shader bằng CSS gradient (đơn giản, tốc độ cao)

## 📝 Ghi chú & Bài học

*   Next.js site có ~48 component files + nhiều thư viện (Three.js, GSAP, Framer Motion, Radix UI, Embla Carousel). Pure HTML migration loại bỏ hoàn toàn dependency tree.
*   Blog cần Phase 2 (Static Site Generator) để export content từ Strapi sang static HTML.
*   Cần kiểm tra visual regression giữa site cũ và mới trước khi swap.
